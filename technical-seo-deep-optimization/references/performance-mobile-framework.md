# Performance and Mobile Framework

## Core Web Vitals workflow

1. Measure by template and segment.
2. Identify dominant bottlenecks (LCP/INP/CLS).
3. Prioritize fixes by impact and rollout complexity.

## Typical improvements

- Image optimization and modern formats
- Critical CSS and script deferral
- Third-party script governance
- Cache/CDN policy improvements
- Rendering pipeline optimization

## Mobile-first checks

- Content parity and render parity
- Touch interaction quality
- Viewport/layout stability
- Resource accessibility on mobile user agents
