# Crawl and Index Playbook

## Crawl diagnostics

- Detect crawl traps and parameter explosions.
- Measure crawl frequency by page type and value tier.
- Find orphan pages and low-value index bloat.

## Indexation controls

- Align robots, meta robots, canonical, and sitemap signals.
- Consolidate duplicate clusters with canonical/redirect rules.
- Restrict thin, filtered, or utility pages from indexing.

## Priority model

Classify URLs into:

- Must-index
- Should-index
- Should-not-index

Then enforce with consistent technical signals.
