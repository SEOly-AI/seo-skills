# Creation and Optimization Standards

## SEO + user optimization

- Match search intent before adding depth.
- Add concrete examples, data, and practical guidance.
- Keep language clear, specific, and non-generic.
- Improve CTR surfaces (title/meta/opening value statement).

## E-E-A-T reinforcement

- Add expert perspective where needed.
- Cite credible references for factual claims.
- Show freshness through updates and version notes.

## AI-writing anti-patterns to avoid

- generic filler with low information density
- repetitive structure with weak differentiation
- unsupported claims or vague recommendations
