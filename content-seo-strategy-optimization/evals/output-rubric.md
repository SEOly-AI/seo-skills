# Content SEO Output Rubric

Use this rubric before final delivery. Mark each as **Pass / Partial / Fail**.

## A. Input and Context Quality (20%)

- [ ] Context files are checked and summarized.
- [ ] Content types, topic focus, competitors, and resources are confirmed.
- [ ] Questions are gap-focused and efficient.

## B. Topic and Cluster Strategy (25%)

- [ ] Topic discovery reflects both keyword demand and user needs.
- [ ] Cluster ownership is clear and avoids overlap.
- [ ] Prioritization is impact-driven and feasible.

## C. Content Planning and Creation Quality (25%)

- [ ] Briefs and outlines are actionable.
- [ ] SEO + user intent alignment is strong.
- [ ] E-E-A-T reinforcement is concrete.

## D. Distribution and Linking Plan (20%)

- [ ] Distribution cadence is realistic.
- [ ] Internal linking strengthens cluster authority.
- [ ] Conversion path is supported by content flow.

## E. Iteration and KPI Model (10%)

- [ ] KPI targets are measurable.
- [ ] Refresh/iteration process is defined.

## Hard Fail Conditions

- Generic strategy without page/topic specificity.
- No internal linking logic.
- No measurable KPI or iteration framework.
