# Outreach and Execution Playbook

## Outreach tracks

1. Editorial pitching
2. Guest contribution with strict quality control
3. Resource page inclusion
4. Data-led linkable asset promotion
5. Partnership-based mentions

## Execution standards

- Personalize outreach context by source audience.
- Map each pitch to one best-fit target URL.
- Keep follow-up cadence structured and respectful.
- Maintain compliance and disclosure standards.

## Anti-spam rules

- No bulk generic blasting.
- No paid link schemes disguised as editorial.
- No manipulative reciprocal linking loops.
