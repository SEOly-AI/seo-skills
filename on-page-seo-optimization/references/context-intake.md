# Context Intake for On-Page SEO

## Read order

1. `.agents/product-marketing-context.md`
2. `.claude/product-marketing-context.md` (legacy)

## Extract these essentials

- Core purpose of the page
- Primary and secondary target queries
- User intent and funnel stage
- Conversion goal for this page
- Brand and compliance constraints

## Gap-only questions

Only ask for missing information:

- Current page URL and page type
- Existing optimization status
- Competitor benchmark URLs
- Success criteria (traffic, ranking, CTR, conversion)

Avoid re-asking details already present in context files.
