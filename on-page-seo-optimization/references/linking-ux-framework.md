# Linking and UX Framework

## Internal linking

- Link from related pages with intent-consistent anchors.
- Support one primary owner page for core terms.
- Use link placement to guide next-step user actions.

## External linking

- Add only high-quality, relevant references.
- Use external links to strengthen trust and factual grounding.

## UX and conversion

- Reduce friction in layout, readability, and CTA path.
- Align CTA type with intent stage.
- Monitor page-level signals tied to engagement and conversion.
