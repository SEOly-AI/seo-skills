# Metadata Optimization Playbook

## Title tag principles

- Place primary intent early.
- Keep language natural and specific.
- Differentiate from close competing pages.
- Avoid duplication and over-optimization.

## Meta description principles

- Summarize value proposition for the query.
- Add a clear motivation to click.
- Reflect page content truthfully.

## Technical checks

- Canonical points to intended page owner.
- Indexing directives match strategy.
- Open Graph/Twitter tags remain aligned (optional but recommended).
