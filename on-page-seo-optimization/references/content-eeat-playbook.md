# Content, Media, and E-E-A-T Patterns

## Content refinement

- Match section depth to query intent.
- Add direct answers before deep detail.
- Remove filler and repeated generic claims.
- Keep one clear topical ownership per page.

## Media optimization

- Use descriptive file names and alt text.
- Add captions where context matters.
- Ensure media supports understanding, not decoration only.

## E-E-A-T reinforcement

- Show author or source credibility where relevant.
- Add evidence, examples, or method transparency.
- Keep content freshness signals clear (updates, versioning, date context).
