# Scenario Checklist

## S1 - Stalled Core Pages

- [ ] Priority page diagnosis completed.
- [ ] Ranking bottlenecks mapped to specific on-page elements.
- [ ] Clear optimization sequence provided.

## S2 - New Page Creation

- [ ] Structure follows best-practice on-page template.
- [ ] Metadata, content blocks, media, and links are preplanned.
- [ ] Conversion path is integrated from day one.

## S3 - Batch Legacy Optimization

- [ ] Unified standards are defined for repeated use.
- [ ] Prioritization model for large page sets is provided.
- [ ] QA checks for consistency are included.

## S4 - SEO + SEM Landing Page Alignment

- [ ] Organic intent and paid conversion intent are jointly addressed.
- [ ] Metadata and CTA strategy avoid message mismatch.
- [ ] Measurement framework includes both SEO and conversion KPIs.
