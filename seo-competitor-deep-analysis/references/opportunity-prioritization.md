# Opportunity and Strategy Prioritization

## Gap exploitation model

1. Match critical fundamentals where behind.
2. Outperform on high-impact, under-defended segments.
3. Differentiate with stronger content angle and user value.

## Prioritization matrix

Each initiative should include:

- expected impact
- implementation effort
- time-to-win
- confidence level
- owner and timeline

## Delivery format

Provide a roadmap with:

- P0 quick wins
- P1 structural improvements
- P2 long-horizon strategic bets
