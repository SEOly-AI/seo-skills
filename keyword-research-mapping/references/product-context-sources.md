# Product Marketing Context Sources

## Preferred file paths

1. `.agents/product-marketing-context.md` — preferred location aligned with agent workflows.
2. `.claude/product-marketing-context.md` — legacy/migration path; read if present.

## Suggested context outline

If the file is missing, mention that it should be added later and ask only for gaps:

- One-sentence company/business positioning
- Core products/services and differentiation
- Target audience profile (roles, pain points, decision chain)
- Primary geo/language markets
- Conversion goals (lead, signup, purchase, booking, etc.)
- Compliance or claim boundaries (if any)

## Link to keyword work

Keyword research should only restate or reference known facts from context. Do not invent business facts or exact SEO metrics. If tools are unavailable, use relative labels like `high/medium/low` or `to be validated`.
