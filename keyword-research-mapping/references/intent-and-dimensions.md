# Search Intent and Analysis Dimensions

## Working intent taxonomy

| Intent Type | User state | Typical query pattern | Common landing format |
|-------------|------------|-----------------------|-----------------------|
| Informational | Learn/understand | what/how/why, tutorials, definitions | guides, blog posts, educational pages |
| Commercial investigation | Compare/evaluate | best, vs, review, alternatives | comparison pages, list pages, category overviews |
| Transactional | Buy/try | price, buy, trial, discount | product pages, pricing pages, checkout funnels |
| Navigational | Find specific brand/page | brand, login, official site | homepage, brand pages, app/login pages |

Mixed intent is allowed. Mark a **primary intent** and **secondary intent**.

## Analysis dimensions

For each keyword/cluster, record:

1. Primary intent and dominant SERP format (article/video/e-commerce/local pack/etc.).
2. Recommended content format: pillar, cluster article, product page, comparison page, tool page, FAQ.
3. Commercial distance to conversion (far/mid/near).
4. Competition/opportunity hypothesis (e.g., UGC-heavy SERP may leave room for expert content).
5. Existing on-site status (for old sites): current ranking URL and intent fit.

## Lightweight competitor mapping

- Check SERP manually or via tools for top ranking competitor URL types.
- Summarize competitor IA level by path pattern (`/blog/`, `/product/`, `/category/`).
- Output the recommended primary page type and differentiation angle (not title-copying).
