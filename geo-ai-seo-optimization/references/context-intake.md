# Context Intake for GEO/AI SEO

## Read order

1. `.agents/product-marketing-context.md`
2. `.claude/product-marketing-context.md` (legacy)

## Extract these fields

- Business model and value proposition
- Core products/services and monetization path
- Target audience segments and buying stage
- Priority markets and language variants
- Existing content inventory by type
- Compliance or claim constraints

## Gap-only questioning rule

If context files are partial, ask only what is missing:

- Target AI search platforms
- Priority funnel stage (top/mid/bottom)
- Content ownership constraints
- Data/tool access for tracking

Never ask for information already present in context.
