# Diagnosis Playbook

## Common symptom patterns

1. Impression drop + stable rankings: likely demand or indexation shifts.
2. CTR drop + stable position: snippet/title mismatch or SERP feature crowding.
3. Position drop + impression drop: content relevance, authority, or technical quality issue.
4. Traffic stable + conversion down: intent mismatch, UX friction, or offer misalignment.

## Root-cause buckets

- Content quality/intention mismatch
- Technical SEO (crawl/index/canonical/performance)
- Authority and competitive pressure
- SERP layout and feature competition
- Conversion flow and page UX

## Confidence levels

Tag each diagnosis as high/medium/low confidence based on available evidence.
