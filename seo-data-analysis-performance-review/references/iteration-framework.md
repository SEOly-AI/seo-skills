# Iteration and Reporting Framework

## Action plan format

Each action should include:

- Priority (P0/P1/P2)
- Owner
- Timeline
- KPI target delta
- Validation method
- Review checkpoint

## Cadence

- Weekly: quick anomaly checks
- Monthly: KPI and action-progress review
- Quarterly: strategic reset and resource reallocation

## Reporting principles

- Keep metrics traceable to source tools.
- Separate observed facts from hypotheses.
- Include decisions and next steps, not only charts.
