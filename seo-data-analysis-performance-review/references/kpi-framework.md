# KPI Decomposition Framework

## Core KPI layers

1. Visibility: impressions, indexed query coverage, ranking distribution
2. Traffic: clicks, sessions, CTR by segment
3. Conversion: CVR, leads/sales/signups from organic
4. Business impact: assisted conversion and revenue contribution

## Standard segment cuts

- Brand vs non-brand
- Page type (homepage/product/category/blog/landing)
- Device and country
- Query intent cluster

## Baseline rules

- Use equivalent comparison windows.
- Annotate major events (site release, campaign, algorithm update) before diagnosis.
