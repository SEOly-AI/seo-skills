# SEO Data Review Output Rubric

Use this rubric before final delivery. Mark each as **Pass / Partial / Fail**.

## A. Data Foundation (20%)

- [ ] Tool access and data scope are clearly documented.
- [ ] Review period and comparison baseline are valid.
- [ ] Segmentation is sufficient for diagnosis.

## B. Analysis Quality (25%)

- [ ] Traffic, ranking, and conversion are analyzed together.
- [ ] Trends and deltas are quantified and contextualized.
- [ ] Facts are separated from assumptions.

## C. Diagnosis Quality (20%)

- [ ] Problems are tied to likely root causes.
- [ ] Confidence level is assigned per diagnosis.
- [ ] Anomalies are clearly explained or escalated.

## D. Opportunity and Action Plan (25%)

- [ ] Opportunities are prioritized by impact/effort.
- [ ] Action items are specific, owner-assigned, and time-bounded.
- [ ] Expected KPI impact and validation method are included.

## E. Iteration and Target Setting (10%)

- [ ] Next-cycle targets are realistic and data-grounded.
- [ ] Follow-up cadence is defined.

## Hard Fail Conditions

- No measurable KPIs in output.
- Purely descriptive report without actions.
- Untraceable conclusions not supported by data.
