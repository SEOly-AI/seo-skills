# Scenario Checklist

## S1 - Routine Monthly/Quarterly/Annual Review

- [ ] Standard KPI decomposition is complete.
- [ ] YoY/MoM/QoQ interpretation is accurate.
- [ ] Next-cycle optimization priorities are explicit.

## S2 - Traffic Anomaly Investigation

- [ ] Drop/spike window is isolated.
- [ ] Candidate causes are ranked by confidence.
- [ ] Immediate mitigation and verification plan is included.

## S3 - SEO Action Impact Validation

- [ ] Pre/post windows are properly defined.
- [ ] Impact attribution is transparent and bounded.
- [ ] Outcome is tied back to original optimization actions.

## S4 - KPI Target Planning

- [ ] Targets are based on historical trend and constraints.
- [ ] Targets include traffic, ranking, and conversion dimensions.
- [ ] Dependencies and assumptions are declared.

## S5 - Team Performance and Resource Review

- [ ] Workstream contribution is quantified.
- [ ] Bottlenecks and resource gaps are identified.
- [ ] Reallocation recommendations are actionable.
