# Penalty Removal Playbook

## Typical penalty drivers

- Manipulative link profile (paid/spam/PBN-like patterns)
- Thin, duplicated, or low-value content at scale
- Over-optimized anchors and keyword stuffing
- Technical misconfigurations causing deindexing or wrong canonical ownership

## Remediation sequence

1. Contain severe issues (robots/index/canonical/redirect disasters)
2. Clean harmful links/content patterns
3. Rebuild quality and trust signals
4. Validate indexing and ranking stabilization

## Validation checks

- Warning status updates in search tools
- Index coverage normalization
- Query cluster ranking stabilization trend
