# Diagnosis Framework

## Classification buckets

1. Algorithmic pressure (core update quality/link reevaluation)
2. Self-inflicted changes (technical/content/architecture)
3. External factors (competitor gains, SERP feature shifts, seasonality)

## Segmentation requirements

- Page type
- Query intent cluster
- Device/country
- Brand vs non-brand

## Evidence model

For each suspected cause, attach:

- observed symptom
- timeline alignment
- supporting metrics/signals
- confidence level (high/medium/low)
